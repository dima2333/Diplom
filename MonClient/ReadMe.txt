Программа предназначена для мониторинга параметров сервера(CPU, RAM, HDD)

Перед использованием обяхательно прочитать инструкцию!

Шаги необходимые для подключение сервера к системе мониторинга:
1 Создать папку MonClient в корне диска C: (C:\MonClient)
2 Рапаковать в эту папку содержимое ZIP архива.
3 В каталоге будут находиться следующие файлы:
- clientAddData.exe : сценарий python, со всеми, необходимыми для работы, модулями.
- clientMon.db : файл базы данных(БД), со структурой данных необходмой для сбора информации об ОС сервера - объекте мониторинга
- sqlite3.exe : утилита для работы с файлом БД.
- MonClient : файл, конфигурации задания(для планировщика заданий), в Windows 10 должен находиться в каталоге: C:\Windows\System32\Tasks. В зависмости от весрии Windows каталоги могут различаться. Для копирования в указанный каталог необходимы права администратора.
- confIPserver.conf : Конфигурационный файл для связи с Web сервером. Необходимо записать IP адрес web-сервера и сохранить.

После распаковки файлов в указанный каталог и настройки планировщика заданий, программа будет запускаться с интервалом в 1 минуту и отправлять данные на сервер мониторинга, где будет обрабатываться и отображаться по адресу:
http://{IP адрес вашего сервера, в формате 255.255.255.255}/index.py

Возможные ошибки:
1 Возможно возникновение ошибки при впыолнении .exe файла на сервере - объекте мониторинга :
"Запуск программы невозможен, так как на компьютере отсутсвует api-ms-win-crt-runtime-l1-1-0.dll. 
Попробуйте переустановить программу".
Эта ошибка возникает в случе запуска тех программ или приложений, которые используют функции 
"Универсальной среды выполнения С" (CRT) Windows 10, а запускаются в предыдущих версиях системы Windows 7, 8, Vista.
Решение:
Необходимо установить ПАТЧ Windows8.1-KB2999226-x64.msu, входящий в архив. Установку проводить с правми администратора.
